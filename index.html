Welcome to CyberSafeLearn

CyberSafeLearn is a small information security awareness site designed for university students.

Everything runs in your browser, and the system never asks for names, emails, or any personal details.

The idea is to learn from short lessons, try a quick quiz, and then leave optional anonymous feedback.

Learn with short lessons

The lessons cover three everyday security topics: phishing awareness, password security, and online privacy.

Each topic is written in simple language so it is easy to skim before you start the quiz.

View lessons

Test yourself with a quiz

The quiz shows one multiple choice question at a time and gives instant feedback on each answer.

At the end you get a score out of ten plus tips on which topics to review.

Start quiz

Leave anonymous feedback

You can send anonymous feedback about the lessons or quiz. No names or emails are collected.

Feedback is sent to a small backend server (see RUN_GUIDE.md to start it).
